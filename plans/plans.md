# X2Y2 API Plans

## Access Model

X2Y2 provides its API via a **free, key-gated beta** model. There is no published tiered pricing structure. Access is obtained by requesting an API key through the X2Y2 Developer Hub (Discord channel).

## API Key Request

- Channel: X2Y2 Developer Hub (Discord)
- Eligibility: Open to developers building web3 applications, analytics tools, and trading bots
- Cost: No publicly disclosed fee

## Notes

- The APIs are in beta; rate limits, request/response formats, and endpoint availability are subject to change without notice
- Maintenance can be scheduled at any time prior to the official release
- Two environments are available: Mainnet and Goerli testnet (now deprecated as Goerli was sunset)
- X2Y2 marketplace shut down on April 30, 2025; smart contracts remain operational

## Status

X2Y2 announced the shutdown of its NFT marketplace effective April 30, 2025. The underlying Ethereum smart contracts continue to function and can still be interacted with programmatically.
